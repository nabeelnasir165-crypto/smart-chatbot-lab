# 🤖 Smart Chatbot Lab

An educational AI pet evolution game for Grade 5 students.

Train your bot, go on missions, and evolve it through 4 stages!

## 🚀 Play Now

👉 **[Live Game](https://YOUR-USERNAME.github.io/smart-chatbot-lab/)**

## 📦 Deploy to GitHub Pages

1. Fork or upload this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to **main branch / root**
4. Done — your game is live!

## 🎮 Features

- 🧠 Train AI with 6 topic modules (48 questions)
- 🚀 6 story missions with mini-games
- 🤖 Bot evolves through 4 stages
- 💬 Chat with your trained bot
- 📊 Performance evaluation screen
- ⏸ Pause / Resume / Volume controls
- 🎓 Visual interactive tutorial
- 📱 Fully responsive (mobile, tablet, desktop)

## 🛠 Tech

- Pure HTML5 / CSS3 / Vanilla JS
- HTML5 Canvas for bot animation
- Web Audio API for all sound effects
- Zero dependencies — single file, works offline
